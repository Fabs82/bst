# Binary Search Tree (BST) Implementation in Ruby

A comprehensive Ruby implementation of a Binary Search Tree data structure with various operations including insertion, deletion, traversal, balancing, and visualization.

## Features

- **Tree Construction**: Build a balanced BST from an array of values
- **Node Operations**: Insert and delete nodes while maintaining BST properties
- **Traversal Methods**:
  - Level-order (breadth-first)
  - In-order (depth-first)
  - Pre-order (depth-first)
  - Post-order (depth-first)
- **Tree Analysis**:
  - Check if tree is balanced
  - Find node depth and height
  - Rebalance unbalanced trees
- **Visualization**: Pretty-print tree structure for easy visualization

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/bst.git
   cd bst
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```
